About this site

  • This is a demo of a national registry, which could be for social protection or programs. If you would like to have the code, please email me: shafique.jamal@gmail.com. I hope to one day put all the code on Github.com
  • This site is implemented in Python using the Django (1.5) framework
  • It easily supports multiple languages
  • It keeps track of changes to social assistance applications

How to use this site

  • Register as a social administrator by filling the form below or clicking register (top right) and filling the form that is then presented
  • After you register, click on "Registry Applications" in the top right menu to view applications (details and index), enter applicaitons, edit applications, and maybe more. Note: you can only edit applications that you have entered, not applications that others have entered. But you can view all applications in the system.
