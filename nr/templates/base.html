About this Site

  • This website is a demonstration website for the free, open-source Social Assistance Registry project. The registry software is useful for social assistance reform projects, which typically require basic application intake software to support electronic intake of social assistance applications. The source code for this site is available here (link to the github.com repository)

  • Please see the screenshots below and watch the YouTube video below to see how one might use this software.

  • This site is implemented in Python using the Django (1.5) framework. It easily supports multiple languages.
